# Ravio Web — Seamless Customer Experience

Ravio (formerly JINISO) is a premium, localized e-commerce web application built using Next.js and Tailwind CSS. It is designed to deliver a seamless shopping experience for premium denim products, bridging online exploration with in-store experiences.

## 🌟 Key Features

- **Omnichannel Denim Experience**: Curated browsing, in-store pickup reservations, and detailed order tracking.
- **Multilingual Support**: High-fidelity translations available in **Indonesian**, **English**, and **Japanese**.
- **Interactive Size Chart**: Visual guides and detailed fitting indicators for tops and bottoms.
- **Virtual Chatbot Concierge**: Automated customer support assistance for tracking orders, sizing help, store information, and human handoff.
- **Admin Dashboard**: Full back-office interface showcasing mock analytics, user log histories, store inventories, and transaction metrics.
- **Responsive Premium UI**: Tailored with beautiful dark and light themes, smooth transitions, and modern glassmorphic elements.

## 🚀 Getting Started

First, install the dependencies and run the development server:

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the live result.

### Available Scripts

- `npm run dev`: Runs Next.js in development mode.
- `npm run build`: Generates an optimized production build.
- `npm run start`: Starts the Next.js production server.
- `npm run lint`: Runs ESLint to check for code quality and style.
